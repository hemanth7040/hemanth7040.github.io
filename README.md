## ⚙️ How It Works

```
git push → GitHub Actions → Hugo Build → Deploy to GitHub Pages
```

Every commit to `main` triggers an automated pipeline:
1. Hugo builds the static site
2. GitHub Actions validates and deploys
3. Live site updates in under 60 seconds — no manual uploads, ever.
